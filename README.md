## Featutes/Highlights
1. A sample e-commerce shopping site <br>
2. View in Shopper or a vendor View <br> 
3. In Shopper view you can:
	* View all Products
	* See details of Induvidual Products
	* ADD to cart 
4. In Vendor view you can:
	* View all the products
	* Add New Product
	* Detete a Product
	* Edit a Product
